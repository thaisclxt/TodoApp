# TodoApp
>A TODO App using Python Flask

</br>

## Installation
https://flask.palletsprojects.com/en/1.1.x/installation/


### Create an envirment
```
py -m venv venv
```

### Activate the envirment
```
cd venv/Scripts/activate.bat
```

### Install Flask
```
pip install flask
```

### Install SQLAclhemy
```
pip install Flask-SQLAlchemy
